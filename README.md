# 논어 (論語) 웹 애플리케이션

공자의 가르침을 담은 논어를 웹에서 쉽게 읽고 탐구할 수 있는 애플리케이션입니다.

## 주요 기능

- 오늘의 구절: 매일 새로운 논어 구절을 랜덤하게 표시
- 장별 보기: 20편의 논어 본문을 장별로 탐색
- 주제별 보기: 덕목과 주제별로 구절 분류
- 한자/한글 병기: 원문과 한글 번역 제공
- 반응형 디자인: 모바일과 데스크톱에서 모두 최적화된 화면

## 기술 스택

- React
- TypeScript
- Material-UI
- Vite

## 시작하기

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 빌드:
```bash
npm run build
```
